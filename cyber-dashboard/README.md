# 🚀 Cyber Dashboard

Футуристическая панель управления с киберпанк дизайном и real-time мониторингом.

## ✨ Возможности

- 🎮 Киберпанк интерфейс
- 📊 Real-time графики и метрики
- 🌐 3D визуализация данных
- 🔥 Неоновые эффекты
- ⚡ WebSocket соединения
- 🎯 Настраиваемые виджеты

## 🚀 Установка

```bash
npm install
npm start
```

## 🎨 Особенности

- Голографические элементы
- Анимированные переходы
- Темная тема с неоновыми акцентами
- Адаптивный дизайн
- Модульная архитектура

## 🔧 Конфигурация

```env
PORT=3000
WEBSOCKET_PORT=3001
THEME=cyberpunk
```
